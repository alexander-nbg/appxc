```{page-status} draft
:summary: Documentation structure is still evolving (2026/03)
```

Intent of pages here is raising awareness of the cross-cutting concerns and connected
features to enable *understanding* the scope of APPXC. Since cross-cutting concerns do not care much about also cross-cutting the APPXC features, this added viewpoint will explains how features relate to each other.

 * possible general motivation via [ISO/IEC
   9126](https://en.wikipedia.org/wiki/ISO/IEC_9126)
